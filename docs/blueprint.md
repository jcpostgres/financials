# **App Name**: Nordico Financials

## Core Features:

- Login and Role Management: Secure login system with admin and receptionist roles, each having specific permissions.
- Point of Sale (POS) System: Manage sales transactions, apply discounts, and finalize payments using various methods.
- Inventory Management: Track products and services, manage stock levels, and categorize items.
- Customer Relationship Management (CRM): Store customer data, track appointment history, and manage customer profiles.
- Financial Reporting: Generate financial reports, track income and expenses, and monitor key performance indicators (KPIs).
- Barber Queue Management: Manage barber turn queue, add/remove barbers, and track service times using Firestore.
- AI-Driven Revenue Forecasts: Use AI to forecast future revenue based on historical data, seasonality, and current trends; tool suggests potential inventory adjustments based on forecasts.

## Style Guidelines:

- Primary color: Saturated gold (#FFC107), to align with the current application's style.
- Background color: Dark gray (#212121), for a modern and professional look, providing contrast against the gold.
- Accent color: Analogous deep orange (#FF8F00) for highlights, call-to-action buttons, and important information.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable, and modern appearance.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a consistent set of icons from Lucide React, with a minimalist design and appropriate weight.
- Subtle transitions and animations for user interactions (e.g., fade-in effects, button hover states) to provide a smooth user experience.
